# Contributing to CloudGuardian AI

Thank you for your interest in contributing to CloudGuardian AI.

## How to Contribute

1. Fork the repository
2. Create a new feature branch
3. Make your changes
4. Commit your changes
5. Push to your fork
6. Submit a Pull Request

## Development Guidelines

- Follow Python best practices
- Write clean and readable code
- Document new features
- Test changes before submitting

## Reporting Issues

If you find a bug or security issue, please open a GitHub Issue with:

- Description of the issue
- Steps to reproduce
- Expected behavior
- Actual behavior

Thank you for helping improve CloudGuardian AI.
