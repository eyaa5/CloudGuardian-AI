## Reporting a Vulnerability

If you discover a security vulnerability in CloudGuardian AI, please report it responsibly.

### How to Report

Create a GitHub Issue describing:

- The vulnerability
- Steps to reproduce
- Potential impact
- Suggested remediation

### Response Process

The project maintainer will:

1. Review the vulnerability report
2. Validate the issue
3. Develop a fix if required
4. Publish an update
